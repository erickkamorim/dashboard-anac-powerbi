# ✈️ Dashboard Executivo | ANAC

## 📊 Sobre o Projeto

Este projeto foi desenvolvido utilizando dados públicos da ANAC para analisar a evolução do transporte aéreo brasileiro entre 2000 e 2022.

O objetivo foi transformar dados em informações estratégicas por meio de um dashboard executivo desenvolvido no Power BI.

---

## 🚀 Tecnologias Utilizadas

- Power BI
- Power Query (ETL)
- DAX
- SQL (Análise Exploratória)
- Figma

---

## 📈 Indicadores

- Passageiros Transportados
- Empresas Aéreas
- Países Atendidos
- Carga Transportada
- Taxa de Ocupação
- Capacidade Ofertada x Demanda (ASK x RPK)

---

## 🎯 Objetivos do Projeto

- Construção de KPIs executivos
- Storytelling com Dados
- Modelagem de Dados
- Visualização de Dados
- Desenvolvimento de Dashboard

---
